How it works
Website generates code via Groq AI, click Insert, it goes to Railway server, Studio plugin polls and receives it, script appears in Studio.

Files needed:
dashboard.html - this file
server.js - Railway backend
StudioBridge_Plugin.lua - Studio plugin
API endpoints
POST /api/ai - generate Lua via Groq
POST /api/command - send to Studio
GET /api/poll/:token - plugin polls
POST /api/result - plugin posts result
GET /api/ping/:token - check connection
