Setup Guide
How NEXUS works
Flow: Type in plain English → NEXUS AI generates perfect Lua → browser sends to Node.js server → Studio plugin polls every 2s → executes in Studio → result in log.

Files needed:
dashboard.html — this file
server.js — Node.js backend
studio-plugin.lua — Roblox plugin
API Endpoints
POST /api/command — send Lua
GET /api/poll/:token — plugin polls
POST /api/result — plugin posts result
GET /api/result/:token — check result
GET /api/ping/:token — heartbeat
