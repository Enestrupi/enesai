Setup Guide
How NEXUS works
100% Free — No API keys ever. NEXUS uses a built-in local rule engine with 150+ Roblox templates. It reads your plain English command, detects every keyword, and combines matching templates into complete working Lua.

Files needed:
dashboard.html — this file
server.js — Node.js backend
studio-plugin.lua — Roblox plugin
API Endpoints
POST /api/command — send Lua
GET /api/poll/:token — plugin polls
POST /api/result — plugin posts result
GET /api/result/:token — check result
GET /api/ping/:token — heartbeat
